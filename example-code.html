1. 加入會員後才可取得API Key

2. 登入會員→會員專區→API認證資訊，取得Token後才具有API使用權

URL

https://10.57.234.147/uswg/stations

使用說明

用縣市代碼取得淹水感知器

Method

Requires Authentication?

Yes

Parameters

countyCode

城市代碼，如B代表臺中市，D代表臺南市

townCode

鄉鎮代碼，例如B05為臺中市北區

API使用範例

http://10.57.234.147/uswg/stations?countyCode=D

http://10.57.234.147/uswg/stations?countyCode=B

http://10.57.234.147/uswg/stations?countyCode=B&townCode=B05
